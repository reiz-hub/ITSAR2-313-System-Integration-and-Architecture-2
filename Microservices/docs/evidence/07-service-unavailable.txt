### Student Service Unavailable (503)

# With Student Service stopped:

curl -i -X POST http://localhost:3000/enrollments ^
  -H "Content-Type: application/json" ^
  -d "{\"studentId\": 1, \"courseId\": 1}"

HTTP Status: 503
Response:
{
  "error": "503 SERVICE_UNAVAILABLE",
  "message": "Student Service is unavailable."
}

---

### Course Service Unavailable (503)

# With Course Service stopped (Student Service running):

curl -i -X POST http://localhost:3000/enrollments ^
  -H "Content-Type: application/json" ^
  -d "{\"studentId\": 1, \"courseId\": 1}"

HTTP Status: 503
Response:
{
  "error": "503 SERVICE_UNAVAILABLE",
  "message": "Course Service is unavailable."
}

---

### Student Service Timeout (504)

# When Student Service exceeds 5 second response time:

curl -i -X POST http://localhost:3000/enrollments ^
  -H "Content-Type: application/json" ^
  -d "{\"studentId\": 1, \"courseId\": 1}"

HTTP Status: 504
Response:
{
  "error": "504 GATEWAY_TIMEOUT",
  "message": "Student Service did not respond in time."
}

---

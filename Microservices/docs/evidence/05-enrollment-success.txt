### Create Enrollment - Success (201)

curl -i -X POST http://localhost:3000/enrollments ^
  -H "Content-Type: application/json" ^
  -d "{\"studentId\": 1, \"courseId\": 1}"

HTTP Status: 201
Response:
{
  "id": 1,
  "message": "created"
}
